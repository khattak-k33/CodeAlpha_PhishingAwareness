# Phishing Awareness Training
**CodeAlpha Cybersecurity Internship — Task 2**

An interactive web-based phishing awareness training module covering:
- 6 types of phishing attacks
- Interactive email dissector (spot 7 red flags)
- URL anatomy and fake website detection
- Social engineering psychology
- 4 real-world case studies
- 10-question knowledge quiz with scoring

## How to Use
Open `index.html` in any web browser — no installation required.
